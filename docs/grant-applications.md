### Project Name
SolTools — Free Non-Custodial Solana Utility Suite for Global Users

### One-line description
A suite of 5 free browser-based Solana tools (rent recovery, token swap, token minting, multi-send, token burn) accessible to anyone with a Phantom/Solflare/Backpack wallet — no sign-up, no servers, no custody.

### What problem does SolTools solve?
Solana wallets accumulate hundreds of worthless spam/airdrop token accounts over time. Each account locks ~0.002 SOL in rent. Non-technical users have no way to recover this value — existing tools require CLI access or charge 50%+ fees. Similarly, minting tokens, sending to multiple wallets, and burning spam tokens all require either dev skills or expensive platforms.

SolTools solves this with a clean, accessible browser interface that works with any major Solana wallet adapter.

### How does SolTools benefit the Solana ecosystem?
- **Reduces friction for new users**: Recovered rent goes back into user wallets, improving the Solana DX
- **100% client-side**: No central server, no custody risk, impossible to rug
- **Multilingual**: EN / FR / AR — expands Solana access to French-speaking (Europe, Africa) and Arabic-speaking (MENA) communities currently underserved by English-only tools
- **Drives on-chain activity**: Every SolSweep, SolMint, SolSend, SolBurn interaction is an on-chain transaction on Solana
- **Open source**: All code on GitHub (yoursearch2008-web/legendary-mind-site), freely forkable

### Why Solana specifically?
The low transaction fees (~$0.0001/tx) make micro-fee tooling viable — charging 0.001 SOL per account closure is only possible on Solana. On Ethereum, gas costs would exceed any reclaimed value. Solana's speed (sub-second finality) also makes the scanning UX smooth.
