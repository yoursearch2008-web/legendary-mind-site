Comment ça marche : chaque compte de token SPL bloque ~0,002 SOL comme loyer. Quand vous ne détenez plus ce token, le compte reste vide mais le SOL reste bloqué. SolSweep trouve tous les comptes vides et les ferme en un clic — le loyer vous revient.

Frais : des frais de 10% sur le SOL récupéré sont inclus dans la même transaction que vous approuvez. Pas de récupération, pas de frais. Vous voyez toujours les montants exacts avant de signer.

Sécurité : 100% dans votre navigateur. Nous construisons uniquement la transaction ; vous l'approuvez dans votre wallet. Nous ne voyons jamais votre phrase de récupération ni votre clé privée. Seuls les comptes vides (solde zéro) sont fermés — vos tokens ne sont jamais touchés.
